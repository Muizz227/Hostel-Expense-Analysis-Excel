# Hostel Expense Analysis using Excel

## Overview
This project analyzes hostel daily expenses using a public Kaggle dataset.
The goal is to understand spending patterns and identify major expense categories
using Microsoft Excel.

## Dataset
- Source: Kaggle (Hostel Life Daily Expense Tracker)
- Format: CSV (raw data)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Analysis Performed
- Category-wise expense aggregation
- Total expense calculation
- Visualization of expenses using charts

## Key Insights
- Food and Snacks are the highest spending categories
- Travel and Health expenses are comparatively lower
- Clear visibility of category-wise spending distribution

## Files in This Repository
- `Hostel_Expense_Tracker.xlsx` — Excel file containing analysis, pivot table, and chart
- `hostel_life_expense_tracker.csv` — Raw dataset used for analysis

## Conclusion
This project demonstrates basic data analysis skills using Excel, including
data summarization and visualization. It is suitable as a beginner-level
data analysis project.
